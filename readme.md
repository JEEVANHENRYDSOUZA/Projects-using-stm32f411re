The aim of this projects was to configure the stm32f411re in order to interface it with various other devices.The devices that I have interfaced are as
1.RGB led
2.ADC for a potentiometer
3.Timers
4.systick interrupts
5.Pulse width Modulation
6.Stepper Motor
7.Dc Motor
8.Encoder
9.Closed Loop Control
